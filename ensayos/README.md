# Ensayos

Guarda aqui los ensayos reales de Los mecanismos invisibles.

No usar esta carpeta para textos ficticios, pruebas editoriales o contenido inventado. Si un ensayo no existe todavia, la pagina debe mostrarlo como pendiente.

## Formato recomendado

Usar un archivo DOCX por ensayo:

```text
nombre-del-ensayo.docx
```

La pagina convierte los DOCX a HTML durante `npm run dev` y `npm run build`.
El primer titulo con estilo de encabezado se usa como titulo visible del ensayo.
