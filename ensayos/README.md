# Ensayos

Guarda aqui los ensayos reales de Los mecanismos invisibles.

No usar esta carpeta para textos ficticios, pruebas editoriales o contenido inventado. Si un ensayo no existe todavia, la pagina debe mostrarlo como pendiente.

## Formato recomendado

Usar un archivo Markdown por ensayo:

```text
nombre-del-ensayo.md
```

Estructura sugerida:

```markdown
# Titulo del ensayo

Pregunta central: ...
Tema: ...
Fecha: YYYY-MM-DD
Firma: Ninan

Texto del ensayo...
```
